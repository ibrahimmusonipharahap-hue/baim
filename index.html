Prestasi Utama

  • Peraih beberapa kali Ballon d'Or (penghargaan pemain terbaik dunia).
  • Juara UEFA Champions League beberapa kali bersama klub-klub papan atas.
  • Pemain kunci tim nasional Portugal dan juara UEFA European Championship (Euro).
  • Pemenang beberapa kali European Golden Shoe (top scorer liga Eropa).
  • Memegang berbagai rekor gol di level klub dan internasional.

Profil Singkat

Nama lengkap: Cristiano Ronaldo dos Santos Aveiro.
Posisi: Penyerang/Winger.
Dikenal dengan kecepatan, teknik, dan ketajaman mencetak gol.

Sorotan Karier

  1. Bersinar di Sporting CP sebagai pemain muda.
  2. Berkembang pesat di Manchester United.
  3. Mencetak rekor di Real Madrid dan meraih banyak gelar.
  4. Bermain untuk klub-klub top Eropa dan tim nasional Portugal.
